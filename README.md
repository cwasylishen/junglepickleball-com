# junglepickleball.com

One-page site for Jungle Pickleball, the indoor pickleball destination in Ojochal, Costa Rica. Four indoor courts, free WiFi, and a community hub.

## Stack

Static HTML with a Tailwind CSS build step. Alpine.js loads from CDN. Hosted on Cloudflare Pages.

- `index.html` - the full single-page site (hero, features, gallery, about, events, FAQ)
- `404.html` - branded not-found page
- `src/input.css` - Tailwind entry plus custom styles
- `assets/styles.css` - compiled, minified Tailwind output (generated by the build)
- `assets/hero/`, `assets/features/`, `assets/gallery/` - image sets
- `tailwind.config.js`, `package.json` - build config
- `_headers` / `_redirects` - Cloudflare Pages config
- `sitemap.xml`, `robots.txt`

## Build

Tailwind scans the HTML and compiles `assets/styles.css`:

```
npm install
npm run build      # one-off build
npm run watch      # rebuild on change during local dev
```

The compiled `assets/styles.css` is committed so the site works without a local build, but Cloudflare regenerates it on every deploy. If you add new Tailwind classes locally, run `npm run build` before previewing.

## Deploy

Connect this repo to Cloudflare Pages with **Build command: `npm run build`** and **Output directory: `.`**. Cloudflare installs dependencies, compiles the CSS, and publishes on every push to `main` (`node_modules` is excluded automatically).

## Local preview

```
npm run build
python -m http.server 8080
```

Then open http://localhost:8080.
